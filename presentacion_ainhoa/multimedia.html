Aspectos a tener en cuenta

  • Hay o no imágenes previas
  • El código es semánticamente correcto o no
  • El código valida o no
  • Tienes en cuenta la dirección de arte o no
  • Puedes/quieres utilizar javascript o no
  • Utilizas un CMS
  • Puedes utilizar componentes de servidor
  • Puedes utilizar componentes de 3ª parte

Elemento <picture>

<picture>
<source media="(min-width: 1100px)" src="large.jpg">
<source media="(min-width: 790px)" src="med.jpg">
<source src="small.jpg">
<!-- fallback -->
<img src="small.jpg" alt="">
texto alternativo/accesible
</picture>

(especificación picture)

srcset

Por ejempo, para retina

<picture>
<source media="(min-width: 1100px)" srcset="large-1.jpg 1x, large-2.jpg 2x">
<source media="(min-width: 790px)" srcset="med-1.jpg 1x, med-2.jpg 2x">
<source srcset="small-1.jpg 1x, small-2.jpg 2x">
<!-- fallback -->
<img src="small.jpg" alt="">
texto alternativo/accesible
</picture>
