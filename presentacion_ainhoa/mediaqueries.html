Responsive Web Design

2) Media Queries

Parámetros

  • width / height
  • device-width / device-height
  • orientation (evitar)
  • aspect-ratio / device-aspect-ratio
  • color / color-index / monochrome
  • resolution

fuente: W3C

Uso

Llamada normal:

<link rel="stylesheet"
type="text/css"
href="estilos.css"
media="screen">


Llamada con media query:

<link rel="stylesheet"
type="text/css"
href="estilos.css"
media="screen and (max-width: 480px)">

Uso

Llamada normal:

.noticia{float:none;}


Llamada con media query:

@media screen and (max-width:480px){
.noticia{float:none;}
}
